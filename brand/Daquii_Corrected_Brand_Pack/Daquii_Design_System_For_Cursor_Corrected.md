# Daquii Corrected Brand Guidelines for Cursor

## Important correction

The authoritative brand direction is the visual reference image:

`daquii_authoritative_brand_reference.png`

The previous abstract D / people logo direction is wrong for this implementation.

Use the island + palm logo from the reference image.

## Brand name

Use lowercase only:

**daquii**

Do not use:

- Daquii
- DAQUII
- Daqui
- Bem Servido

## Brand idea

Daquii means "from here".

The platform connects visitors with trusted local people and services.

The brand is about people and services from here, not places on a map.

## Tagline

**Tudo o que você precisa, daqui.**

## Hero headline

**Pessoas de daqui, para você.**

## Logo system

Use the corrected logo assets:

- `daquii_logo_mark_corrected.svg`
- `daquii_logo_horizontal_corrected.svg`

### Logo mark

The mark must be:

- organic island outline
- palm tree inside
- bright gradient outline
- teal palm tree
- no external dot

### Wordmark

The wordmark must be:

- lowercase `daquii`
- small d
- Deep Navy
- two coloured i dots: Ocean Teal and Tropical Pink

## Forbidden logo directions

Do not use:

- abstract D with people
- handshake icon
- location pin
- capital D
- exterior dot beside the island outline
- dark neon Miami Vice styling

## Colour palette

| Colour | Hex | Usage |
|---|---|---|
| Tropical Pink | #FF2D6D | Primary pop accent, emphasis, badges |
| Sunset Orange | #FFB62B | Warm accent, gradient midpoint |
| Ocean Teal | #00C2BB | Primary CTA, active state, palm, links |
| Deep Navy | #0B1D3A | Wordmark, headings, text, nav |
| Sand | #FFF7ED | Warm backgrounds and cards |
| White | #FFFFFF | Main background |
| Soft Border | #E7EEF2 | Borders and dividers |
| Muted Text | #5B677A | Supporting copy |

## Colour rule

Make the colours pop.

Do not mute the palette.

Do not make the site grey or overly pastel.

## Brand gradient

```css
linear-gradient(135deg, #FF2D6D 0%, #FFB62B 48%, #00C2BB 100%)
```

Use this mainly for:

- logo mark
- premium badges
- small dividers
- selected states

## Typography

Use:

- Headings: Sora or Montserrat
- Body: Sora or Inter

Headings should feel rounded, modern and confident.

## Layout style

The UI should feel like:

Airbnb x premium tropical concierge.

Use:

- white backgrounds
- sand panels
- large rounded cards
- generous spacing
- bright accents
- bold navy typography
- subtle shadows
- mobile-first layouts

## Provider card rules

Provider cards should emphasise real people.

Show:

1. Large face photo
2. Provider name
3. Category
4. Languages
5. Half-day price
6. Trust badges
7. WhatsApp CTA

## Provider profile rules

Above the fold:

1. Large provider photo
2. Name
3. Category
4. Trust badges
5. WhatsApp button
6. Intro video if available

## Advertising rules

Advertising should feel native and premium.

Use:

- clean sponsor cards
- Sand backgrounds
- logo or image
- short description
- clear CTA
- `Patrocinado` label where appropriate

## Admin dashboard styling

Keep it practical but on-brand.

Use:

- white background
- Sand panels
- Deep Navy headings
- Ocean Teal CTAs
- Tropical Pink warnings
- Sunset Orange pending states

## Provider dashboard styling

Simple, friendly and checklist-based.

Use profile completion cards and clear next steps.

## Illustration style

Use bright daytime tropical illustrations similar to the reference.

Show real service moments:

- driver greeting visitor
- chef preparing food
- cleaner preparing villa
- massage therapist
- boat captain
- babysitter
- grocery delivery

Avoid:

- dark night scenes
- neon nightclub style
- generic stock visuals
- childish cartoons
- excessive palm trees
